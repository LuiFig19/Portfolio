# 🤖 Automated Setup - Just Follow These Steps

I'll do everything I can automatically. You just need to:
1. Install Git (one-time, 2 minutes)
2. Create one repository on GitHub (one-time, 30 seconds)
3. Sign in when prompted (one-time, 30 seconds)

Then I'll handle the rest!

---

## Step 1: Install Git (REQUIRED - 2 minutes)

**You must do this once:**

1. Download: https://git-scm.com/download/win
2. **Run the installer**
3. **Use all default settings** (just keep clicking Next)
4. **IMPORTANT:** After installation, close this terminal and reopen it

**After installing, come back and I'll do everything else!**

---

## Step 2: Create GitHub Repository (REQUIRED - 30 seconds)

1. Go to: https://github.com/new
2. **Repository name:** `portfolio` (or any name you want)
3. Make it **Public**
4. **DO NOT** check any boxes (no README, no .gitignore, nothing)
5. Click **Create repository**

**Don't close that page - you'll need the URL in a second!**

---

## Step 3: Tell Me Your Info

Once Git is installed and repo is created, tell me:
- Your GitHub username
- The repository name you created (probably `portfolio`)
- Your name (for Git commits)
- Your email (the one on your GitHub account)

**Then I'll run all the commands automatically!**

## What I'll Do Automatically:

✅ Initialize Git repository

✅ Add all your files

## Step 4: After I Push to GitHub

1. Go to: https://vercel.com
2. **Sign up with GitHub** (one click)
3. Click **Add New...** → **Project**
4. Find your repository
5. Click **Import** → **Deploy**

**Vercel handles everything else automatically!**

---

## Step 5: Add Your Domain

1. In Vercel project → **Settings** → **Domains**
2. Enter: `luifigueroa.com`
3. Vercel shows you exactly what DNS records to add
4. Add them at your domain registrar
5. Wait 5-30 minutes

**Done!**
